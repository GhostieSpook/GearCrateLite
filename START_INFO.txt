================================================================================
GEARCRATE - START OPTIONEN
================================================================================

Es gibt jetzt VIER Möglichkeiten, GearCrate zu starten:

────────────────────────────────────────────────────────────────────────────────
1. start-browser.bat (Normal)
────────────────────────────────────────────────────────────────────────────────
   ✓ Startet HTTP Server
   ✓ Öffnet Browser automatisch
   ✓ Keine Admin-Rechte
   ✗ InvDetect Scanner funktioniert NICHT

   Verwende dies für:
   - Normale Inventar-Verwaltung
   - Gear Sets Verwaltung
   - Bulk Import von Cornerstone

────────────────────────────────────────────────────────────────────────────────
2. start-browser-admin.bat (EMPFOHLEN für Scanner)
────────────────────────────────────────────────────────────────────────────────
   ✓ Startet HTTP Server mit Admin-Rechten
   ✓ Öffnet Browser automatisch
   ✓ InvDetect Scanner funktioniert VOLLSTÄNDIG
   ✓ Alle Features verfügbar

   Verwende dies für:
   - InvDetect Scanner (Import from SC)
   - Vollständige Funktionalität
   - Empfohlen für tägliche Nutzung

────────────────────────────────────────────────────────────────────────────────
3. start-desktop.bat (Normal)
────────────────────────────────────────────────────────────────────────────────
   ✓ Startet Desktop-Fenster (pywebview)
   ✓ HTTP Server im Hintergrund
   ✓ Keine Admin-Rechte
   ✗ InvDetect Scanner funktioniert NICHT

   Verwende dies für:
   - Desktop-App Gefühl
   - Normale Inventar-Verwaltung ohne Scanner

────────────────────────────────────────────────────────────────────────────────
4. start-desktop-admin.bat (Desktop + Scanner)
────────────────────────────────────────────────────────────────────────────────
   ✓ Startet Desktop-Fenster mit Admin-Rechten
   ✓ InvDetect Scanner funktioniert VOLLSTÄNDIG
   ✓ Alle Features verfügbar

   Verwende dies für:
   - Desktop-App mit Scanner-Funktionalität
   - Alternative zu Browser-Modus

================================================================================
WARUM ADMIN-RECHTE?
================================================================================

Der InvDetect Scanner benötigt Admin-Rechte für:
   - Tastatur-Hooks (INSERT/DELETE Tasten)
   - Fenster-Switching (zwischen GearCrate und Star Citizen)
   - PyAutoGUI Maus-Steuerung

Ohne Admin-Rechte:
   ✗ Scanner kann nicht gestartet werden
   ✗ "Import from SC" Feature ist nicht nutzbar
   ✓ Alle anderen Features funktionieren normal

================================================================================
EMPFEHLUNG
================================================================================

Für tägliche Nutzung MIT Scanner:
   → start-browser-admin.bat (oder start-desktop-admin.bat)

Für schnelle Nutzung OHNE Scanner:
   → start-browser.bat (oder start-desktop.bat)
